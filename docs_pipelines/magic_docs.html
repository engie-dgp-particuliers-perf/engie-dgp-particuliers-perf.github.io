docs.magic_docs.magic_docs.extract_pipelines_from_configs(pipelines: list, pipelines_path: str) → dict¶

extract all pipelines and their jobs from their configs

To be used in https://engie-dgp-particuliers-perf.github.io/docs_pipelines/pipelines.html. It also removes jobs not present in jenkinsfiles.

Parameters

  • pipelines (list) – list of pipeline names

  • pipelines_path (str) – relative path to pipelines folder (e.g. : ‘../../pipelines’)

Returns

list of all pipelines with their jobs. Example :
>>> {
>>>     "ga_traitements": {  # pipeline
>>>         "t_sessions": {  # job
>>>             "sources": [("BigQuery", "23435435426.t_sessions")],
>>>             "target": ("BigQuery", "ga.t_ga_sessions"),
>>>         },
>>>         ...
>>>     },
>>>     ...
>>> }

Return type

dict
